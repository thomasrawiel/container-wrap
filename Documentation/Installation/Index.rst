============
Installation
============

Install this extension via composer::

    composer require traw/container-wrap

or add it to your extension's composer.json file::

    "require": {
        "typo3/cms-core": "^12 || ^13",
        "traw/container-wrap": "^1.0"
    }

No additional configuration needed.
